# Simple E-commerce

## Installation

1. Clone the repository:

    ```
    git clone https://github.com/ahmed5353/simple-ecommerce.git
    ```

2. Navigate to the project directory:

    ```
    cd simple-ecommerce
     ```
3. Install dependencies:

    ```
    npm install
    ```

4. Start the development server:

    ```
    npm run dev
    ```
